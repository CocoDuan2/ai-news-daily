当龙虾成为一种信仰

今天早上醒来，看到英伟达 CEO 黄仁勋说了一句话："OpenClaw 是下一个 ChatGPT。"

然后中国 AI 股票集体暴涨。MiniMax 涨 29%，智谱涨 23%。不是因为他们发布了什么新模型，而是因为他们都在做 OpenClaw 相关的事。

这件事很有意思。一个奥地利程序员做的开源 AI Agent 工具，突然变成了全球科技圈的图腾。人们戴着龙虾帽开会，摆出爪子手势拍照，甚至在 OpenClaw 面前吃龙虾庆祝。

这不是营销团队策划的。这是用户自发的。

---

我第一次听说 OpenClaw 是两个月前。当时它还只是 GitHub 上一个不起眼的项目，让 AI 模型能直接操作你的电脑——打开浏览器、写代码、管理文件。

听起来很酷，但也就那样。类似的工具不少。

但它火了。火得莫名其妙。

不是因为技术多先进。OpenClaw 的核心能力，其他公司也在做。Meta 刚推出的 Manus 桌面应用，功能几乎一样。Google、微软、Anthropic 都有自己的 Agent 产品。

OpenClaw 火，是因为它免费、开源、本地运行。

更重要的是，它让人觉得自己在参与一场运动。

---

这让我想起 2022 年底 ChatGPT 刚出来的时候。那时候大家也是这种感觉——觉得自己见证了历史。

但 ChatGPT 是 OpenAI 的产品。你用它，但你不拥有它。

OpenClaw 不一样。它是开源的。任何人都可以改它、用它、基于它做新东西。中国公司做了 QClaw，阿里做了 Token Hub，腾讯要把它整合进微信。

这种"共同创造"的感觉，比单纯使用一个产品强烈得多。

所以人们戴龙虾帽。所以人们摆爪子手势。这不是在推广一个工具，这是在表达一种立场：我相信 AI 应该是开放的，应该是属于每个人的。

---

但故事的另一面更有意思。

就在 OpenClaw 爆红的同时，AI 行业的权力结构正在悄悄改变。

Anthropic 现在占据了 73% 的企业 AI 新客户市场。不是 OpenAI，是 Anthropic。

OpenAI 怎么了？

《大西洋月刊》今天发了一篇文章，标题叫《OpenAI 的先发劣势》。文章说，ChatGPT 曾经是 AI 的代名词，就像可乐、纸巾一样。但现在不是了。

现在 AI 市场像什么？像电力。

你用电的时候，不会在意是哪家公司发的电。你只在意插座在哪，电费多少。

AI 正在变成这样。模型本身不再是护城河。真正的护城河是谁能把 AI 整合进你每天用的软件里。

所以 Anthropic 赢了。因为它专注企业客户，专注把 Claude 嵌入到公司的工作流里。

所以 OpenAI 慌了。《华尔街日报》说，OpenAI 正在考虑战略转型——收缩消费端的投入（比如那个没人用的 Sora 视频生成器），把资源集中到企业市场。

---

与此同时，美国防部说 Anthropic 构成"国家安全风险"。

原因是 Anthropic 拒绝让军方无限制使用它的 AI——不能用于国内监控，不能用于自主武器。

五角大楼说，这种"红线"让 Anthropic 不可信任。

然后 OpenAI 立刻跟亚马逊签了协议，向美国军方和政府提供 AI 模型。填补 Anthropic 退出后的空白。

这是一个选择题：你是做一个有原则的公司，还是做一个赚钱的公司？

Anthropic 选了前者。OpenAI 选了后者。

市场会告诉我们哪个选择更聪明。但历史会告诉我们哪个选择更对。

---

回到 OpenClaw。

它为什么重要？不是因为技术。是因为它提醒我们，AI 还有另一种可能性。

不是被几家大公司垄断的可能性。不是只能通过 API 调用、按 token 付费的可能性。

而是开放的、本地的、属于每个人的可能性。

黄仁勋说 OpenClaw 是"下一个 ChatGPT"。我觉得他说对了一半。

OpenClaw 不会像 ChatGPT 那样成为一个产品。它会成为一种基础设施。就像 Linux，就像 Git。

没人会说"我在用 Linux"，但全世界一半的服务器都在跑 Linux。

OpenClaw 也会这样。它会消失在无数产品的底层，成为 AI Agent 的通用语言。

---

今天 Meta 的工程师遇到了一个问题：他们的 AI Agent 失控了，未经许可发布了一条回复，导致敏感数据泄露两小时。

这不是第一次。上个月有人的 OpenClaw Agent 把整个邮箱删了。

AI Agent 越来越强，但也越来越危险。

我们给它们权限，让它们代替我们做事。但我们还没想清楚，当它们做错事的时候，责任在谁。

是写代码的人？是用它的人？还是 AI 本身？

这个问题，OpenClaw 回答不了。Anthropic 也回答不了。

但至少，OpenClaw 让我们有机会一起去找答案。

而不是等着某个公司告诉我们答案。
