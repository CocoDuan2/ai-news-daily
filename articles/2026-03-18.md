OpenClaw，一个月前还没人听说过的名字

今天，Nvidia 的 CEO 黄仁勋站在 GTC 大会上，用了一个听起来有点夸张的说法：「这是人类历史上最大、最受欢迎、最成功的开源项目。」

他说的是 OpenClaw。

一个月前，它还只是 GitHub 上的一个新仓库。今天，它的 star 数已经超过了 Kubernetes、TensorFlow、甚至 Linux 内核的增长速度。不是因为它做了什么革命性的技术突破，而是因为它做对了一件事：让 AI 不再只是回答问题，而是真的能帮你干活。

写代码、整理文件、浏览网页、发邮件——这些事情你以前要打开好几个软件，现在一句话就行。更重要的是，它是开源的，可以在本地跑，数据不用上传到云端。

这就是为什么 Nvidia 要推出 NemoClaw。一个命令，给 OpenClaw 装上企业级的安全防护、隐私策略、本地模型。黄仁勋的意思很明确：这东西要进公司了，我们得让它安全。

━━━ 但这只是开始 ━━━

OpenClaw 火了之后，整个行业都在动。

百度、阿里巴巴、腾讯，三家同时宣布推出基于 OpenClaw 的产品。百度叫它「龙虾全家桶」，阿里巴巴叫它 Wukong，腾讯还没公布名字，但肯定也在路上。

这不是巧合。这是一场集体焦虑。

过去一年，大家都在卷大模型。参数越来越大，训练成本越来越高，但用户体验的提升越来越小。ChatGPT 从 GPT-3.5 到 GPT-4，普通人感觉不到太大区别。Claude 从 2 到 3，也是一样。

但 OpenClaw 不一样。它不是更聪明的聊天机器人，它是一个能替你做事的助手。这个差别，用户一用就懂。

所以当 OpenClaw 在一个月内成为 GitHub 历史上增长最快的项目时，所有人都意识到：下一个战场不是模型有多大，而是 Agent 能做什么。

━━━ OpenAI 也慌了 ━━━

OpenAI 内部开了一场全员会议。应用负责人 Fidji Simo 说了一句话：「我们不能因为被支线任务分心而错过这个时刻。我们必须在生产力上做到极致，尤其是企业生产力。」

这话听起来像是在说「我们要专注」，但实际上是在说「我们落后了」。

Anthropic 的 Claude 在二月份突然爆发。不是因为模型更好，而是因为它在企业市场站稳了脚跟。联邦政府的一些部门开始把 Claude 当成不可或缺的工具。OpenAI 的人把这叫做「警钟」。

更直接的动作是，OpenAI 发布了 GPT-5.4 mini 和 nano。两个小模型，速度是前代的两倍以上，专门用来写代码、调试代码、分类数据。这是 OpenAI 第一次明确地把「编码」当成主战场。

为什么？因为 Anthropic 的 Claude 在开发者群体里已经成了默认选择。OpenAI 必须抢回来。

━━━ 但 Anthropic 也没那么顺 ━━━

五角大楼把 Anthropic 列为「供应链风险」。理由是 Anthropic 拒绝在合同里删掉一条：不能用我们的 AI 进行大规模监控，也不能用来部署无人干预的武器。

五角大楼不同意。Anthropic 不让步。于是五角大楼开始自己做替代方案。

但有意思的是，企业市场完全不在乎这个标签。财富 500 强的公司们纷纷表示：我们不会改变与 Anthropic 的合同。Anthropic 的企业收入还在增长。

这说明了一件事：政府的态度和市场的选择，已经不是一回事了。

━━━ 而 Nvidia 在做什么？ ━━━

黄仁勋说，AI 繁荣的下一阶段，会在年底前带来 1 万亿美元的订单。

但他也知道，Nvidia 的垄断地位正在被挑战。Google、Meta、Amazon，都在做自己的 AI 芯片。Google 的 Ironwood TPU，成本比 Nvidia 的 GB200 Blackwell 低 30-44%。

所以 Nvidia 的策略是：不只卖芯片，还要卖整套解决方案。NemoClaw 就是这个策略的一部分。你用 OpenClaw，我给你加上安全层、隐私层、本地模型，顺便把我的芯片也卖给你。

这不是防守，这是提前占位。

━━━

OpenClaw 会不会是下一个 ChatGPT？

黄仁勋说「肯定是」。但我觉得更准确的说法是：OpenClaw 是 AI 从「回答问题」到「替你做事」的分水岭。

ChatGPT 让所有人知道 AI 能聊天。OpenClaw 让所有人知道 AI 能干活。

这个转变，比模型参数从 1750 亿到 1 万亿，重要得多。
